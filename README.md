ASM Sentinel is a professional Attack Surface Management platform that continuously discovers, monitors, and assesses your organization's external attack surface.

## 🚀 Features

- **Asset Discovery** - Find domains, subdomains, and IPs
- **Port Scanning** - Discover exposed services and ports
- **Technology Detection** - Identify software stacks and versions
- **SSL/TLS Monitoring** - Track certificate expiry and weak ciphers
- **Risk Scoring** - Automated risk assessment and prioritization
- **Real-time Dashboard** - Web-based monitoring interface
- **Automated Reports** - Daily HTML reports with actionable insights
- **Change Detection** - Track new assets and configuration changes
